# Vapour

A Steam-inspired game marketplace website built using HTML and CSS

## Features
- Responsive design
- Game catalog page
- Navigation menu
- Search functionality
- Interactive UI elements

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Download the project.
2. Open index.html in a browser.
